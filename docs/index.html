Powerful Open Source Virtualization

oVirt is a free open-source virtualization solutions for your entire enterprise.

Download 4.2.7 Quickstart guide

Community power

oVirt is an open-source distributed virtualization solution, designed to manage your entire enterprise infrastructure. oVirt uses the trusted KVM hypervisor and is built upon several other community projects, including libvirt, Gluster, PatternFly, and Ansible.

What's included

Rich User Experience

Web-based user interfaces for both administrator and non-administrator users.

Integrated Management

Manage hosts, storage, and network configurations seamlessly.

Live Migration Capabilities

Migrate virtual machines and disks between hosts and storage in real time.

High Virtual Machine Availability

Plenty of virtual machines to choose from in the event of a host failure.
